# 🛒 Shopping Cart

A simple **Shopping Cart Web Application** built using **HTML, CSS, and
JavaScript**. This project demonstrates the core functionality of an
e‑commerce cart where users can browse products, add items to the cart,
update quantities, and view the total price dynamically.

------------------------------------------------------------------------

# 🚀 Features

-   📦 Display products dynamically
-   ➕ Add items to cart
-   ➖ Remove items from cart
-   🔢 Update item quantities
-   💰 Automatically calculate total price
-   🛍️ View selected items in cart
-   ⚡ Fast and lightweight frontend implementation

------------------------------------------------------------------------

# 🛠️ Tech Stack

-   **HTML5** -- Structure of the website
-   **CSS3** -- Styling and layout
-   **JavaScript (Vanilla JS)** -- Cart logic and dynamic updates

------------------------------------------------------------------------
